##
STEP 1: Clone the project (Only once)

Ask your team leader for the repository URL.

git clone https://github.com/Prince9724/CBS-core-banking-system-.git

Go inside the project.

cd CBS-core-banking-system-
STEP 2: Check which branch you're on
git branch

Example:

* master
STEP 3: Create your branch (Only once)

If your name is Sachin:

git checkout -b sachin

Check:

git branch

Output:

master
* sachin
STEP 4: Push your branch to GitHub (Only once)
git push -u origin sachin

Now your branch exists on GitHub.
